# JSON 配置

ThreeJSON 场景文件通常包含两个主要部分：

- `sceneConfig`：渲染器、相机、控制器、灯光、背景、环境、辅助对象和渲染循环设置。
- `worldInfo`：按类型化列表组织的场景对象，例如盒子、球体、模型、精灵、面板和 domain 记录。
