# Download

## Runtime

Install ThreeJSON from npm:

```bash
npm install threejson
```

Use it from modern ESM code:

```js
import { createJsonScene } from "threejson/core";
```

## Optional Assets

Demo assets can be installed separately when available:

```bash
npm install @threejson/assets
```

The assets package is intended for examples, textures, sample JSON scenes, and local documentation resources. Production applications can use their own asset pipeline and point ThreeJSON JSON fields to those URLs.
