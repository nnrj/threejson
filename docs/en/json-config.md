# JSON Configuration

ThreeJSON scene files normally contain two major parts:

- `sceneConfig`: renderer, camera, controls, lights, background, environment, helpers, and render loop settings.
- `worldInfo`: scene objects grouped by typed lists such as boxes, spheres, models, sprites, panels, and domain records.
